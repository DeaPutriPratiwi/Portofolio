# PORTOFOLIO DESAIN  
**Dea Putri Pratiwi**

Desainer grafis profesional yang menyediakan layanan desain untuk kebutuhan
umum dan bisnis. Seluruh proses dikerjakan secara mandiri dengan fokus pada
kualitas, ketelitian, dan kepuasan klien.

---

## LAYANAN
- Desain & edit foto
- Editing video
- Desain logo (2D & 3D)
- Desain kemasan produk
- Branding bisnis
- Restorasi foto
- Desain baju / apparel
- Konten media sosial

---

## TOOLS YANG DIGUNAKAN
Lightroom, Photoshop, Snapseed, Canva, CapCut, dan tools desain lainnya.
